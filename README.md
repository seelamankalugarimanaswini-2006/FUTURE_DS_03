# FUTURE_DS_03

## Marketing Funnel & Conversion Analysis

## Description
This dashboard analyzes the marketing funnel performance and conversion rates across different stages and channels.

## Tools Used
- Power BI

## Files Included
- Power BI Dashboard (.pbix)
- Dataset
- Dashboard Screenshot

## Key Insights
- Total Leads: 514
- Conversion Rate: 19.65%
- Drop-off is highest at early funnel stages
- Some channels generate high leads but low conversions
- High-performing channels contribute more to final customers
